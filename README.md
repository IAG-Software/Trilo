# Trilo - AI-Powered Personal Productivity

Trilo is a high-performance desktop application for managing notes and projects with AI-assisted features.

## Tech Stack
- **Frontend**: Next.js 16+, NextUI, Framer Motion, Tailwind CSS
- **Backend**: Python (SQLite, electron-cgi)
- **Desktop**: Electron

## Prerequisites
- [Node.js](https://nodejs.org/) & [pnpm](https://pnpm.io/)
- [Python 3.x](https://www.python.org/)

## Getting Started

1. **Install Frontend Dependencies**:
   ```bash
   pnpm install
   ```

2. **Install Python Dependencies**:
   ```bash
   pip install -r backend/requirements.txt
   ```

3. **Run in Development**:
   ```bash
   pnpm electron:dev
   ```

## Key Features
- **Notes**: Obsidian-style minimalist writing environment.
- **Operations Hub**: Manage multiple projects in one place.
- **Trilo Board**: Flexible Trello-style Kanban boards with drag-and-drop.
- **AI Integration**: AI-powered task breakdown and note summarization.

## Aesthetics
The app features a "Midnight Neon" theme with glassmorphism, vibrant gradients, and smooth animations powered by Framer Motion.
